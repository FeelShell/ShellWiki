# 通配符

- ?      匹配一个任意字符
- *      匹配任意内容
- []     匹配中括号中的任意一个字符
- [-]    匹配中括号中任意一个字符， - 代表一个范围 A-Z 0-9
- [^]    匹配不是中括号内的一个字符 [^0-9]

适用于文件名或目录的匹配

# 其他特殊符号

- '' 在单引号中所有特殊符号，都没有特殊含义
- "" 在双引号中特殊符号都没有特殊含义，但是 $ ` \ 是例外， 拥有调用变量的值、引用命令、转义符的含义
- `` 反引号括起来的内容是系统命令，在Bash会先执行它，和$() 作用一样，由于容易看错，推荐使用 $()
- $()和反引号一样，用来引用系统命令
- #  注释
- $  调用变量的值
- \  转义
